# school-projects
My school projects, utilizing ML, data preproccesing and statistics.


## statistics
[Spain employment](prs-spain-employment.pdf) - Geographic and sector-based employment study in Spain (NUTS3 regions).

[EU Salary Analysis](prs-sat-score-analysis.pdf) in collaboration Šimon Kubeš - A statistical study using R and Eurostat data to model the impact of GDP, EU membership and other factors on average wages using linear regression.
